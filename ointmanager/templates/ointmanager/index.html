Welcome to Oints!

Oints is the home for tokenized marketing on Base.

Oints are company loyalty ERC20 tokens that are earned through company purchases or affiliate referrals.
They can be redeemed for company purchases or traded (via Uniswap or other means) for Eth or tokens.
Holders redeem company oints by burning them.

We are running as a demo on Base Sepolia. Our GitHub repo is here.

API docs

The Oints API can be used to:

  1. Mint company oints to a Base address
  2. Check current oint balances

Checking a Balance

To check a wallet's balance of some oint, navigate to oints.xyz/api/checkbalance/{str:oint_id}/{str:address},
 replacing {str:oint_id} with the oint_id string for that oint and
 replacing {str:address} with the Base wallet address that you want to check

Fill in the blanks and you can check my wallet address for TestCoints by navigating to:
 oints.xyz/api/checkbalance/e38f03ba-fa00-4349-be96-200788c0aad6/0x27a3E9624B31C0b2D6841761A0e8f285B32977bb

Minting an oint

To mint your company's oint to some address, navigate to oints.xyz/api/mint_oints_to_address/{str:oint_id}/{str:address}/{int:amount}/{str:company_api_key},
 replacing {str:oint_id} with the oint_id string for that oint,
 replacing {str:address} with the Base wallet address to which you want to mint,
 replacing {int:amount} with the amount of oints you want to mint, and
 replacing {str:company_api_key} with your company's api key
